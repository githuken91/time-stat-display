# Time Stat Display

A simple mod that displays the time of day in the bottom-left corner.

## Why use this mod?

Although there may be other mods doing similar things, this mod is very simple and fulfills its designed purpose. Being a superior version of the ingame clock item. The clock item only tells you if it's day or night, whereas, with this mod, you know the time of day down to 5 minute intervals.

# Installation

## Modrinth App (recommended)

Look for the mod in the Modrinth app, and install it to the 26.1 profile of your choice.

This will automatically install the required dependency (YACL) with the mod.

## Github/Modrinth

Start by downloading the mod .jar from either the [releases page](https://github.com/githuken91/time-stat-display/releases/latest) or the [modrinth page](https://modrinth.com/mods/time-stat-display).

Then, go to Modrinth and download the respective [YACL version](https://modrinth.com/mod/yacl/versions) for the TSD version you downloaded.


|        TSD         |     YACL     |
|:------------------:|:------------:|
|     1.0-1.1.2      |  3.9.3-26.1  |
| 1.1.3 (unreleased) |  3.9.3-26.2  |
